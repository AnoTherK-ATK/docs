## Quiz Example

This page demonstrates the Quiz component with examples of both multiple choice and multiple answer questions.

## How to Use

To use the Quiz component in your MDX files:

```astro
import Quiz from '../../components/Quiz.astro';

<Quiz 
  title="Your Quiz Title"
  questions={[
    {
      id: 'q1',
      type: 'single', // or 'multiple'
      question: 'Your question here',
      options: [
        { id: 'a', text: 'Option 1' },
        { id: 'b', text: 'Option 2' },
        { id: 'c', text: 'Option 3' },
        { id: 'd', text: 'Option 4' }
      ],
      correctAnswers: ['b'],
      explanation: 'Optional explanation'
    }
  ]}
/>
```

## Features

- ✅ **Multiple Choice**: Single answer questions with radio buttons
- ✅ **Multiple Answer**: Questions with multiple correct answers (checkboxes)
- ✅ **Auto-scoring**: Automatic score calculation when submitted
- ✅ **Visual Feedback**: Color-coded correct/incorrect answers
- ✅ **Explanations**: Optional explanations for each question
- ✅ **Mobile Friendly**: Responsive design for all devices
